CHAPITRE 1 : STATISTIQUE DESCRIPTIVE SIMPLE

La statistique est la science dont l'objet est de recueillir, de traiter et d'analyser des données issues de l'observation de phénomènes aléatoires. L'analyse des données est utilisée pour décrire les phénomènes étudiés, faire des prévisions et prendre des décisions à leurs sujet. En ce sujet la statistique est un outil essentiel pour la compréhension et la gestion des phé- nomènes complexes. La statistique descriptive a pour but de résumer l'information contenue dans les données de façon synthétique et efficace. Elle utilise pour cela des représentations de données sous forme de graphiques, de tableaux et d'indicateurs numériques (par exemple des moyennes). Elle permet de dégager les caractéristiques essentielles du phénomène étudié. On parle de statistique descriptive à une dimension (univariée), si les donn ées ne sont relatives qu'à une seule variable. Dans le cas où l'on s'intéresse à deux variables simultanément, on parle alors de statistique bivariée. Si l'ensemble des données provient de l'observation de plusieurs variables, on doit faire appel aux méthodes statistiques multivariées.

1/ Généralités

1.1/ Population :

On appelle population un ensemble fini noté Ω tel que :

Ω= {ω1 , ω2 , ω3 ......,ωN}. N est la taille de la population. Les éléments ωi(i = [1;N]) sont appelés individus ou unités statistiques.

1.2/ Echantillon :

On appelle échantillon un sous ensemble de la population de taille n, (n < N).

Example :

On s'intéresse à l'ensemble des étudiants de l'U.S.T.H.B.

(a) Population : les étudiants de l'U.S.T.H.B.

(b) Individu : un étudiant de l'U.S.T.H.B.

(c) Échantillon : un sous ensemble de 100 étudiants.

2/Caractères (variables statistiques), modalités :

2.1/ Caractère (variable statistique)

Un individu donné de la population peut être étudié selon certaines propri étés. Ces propriétés sont appelées caractères ou variables statistiques. On note X (ou Y ) le caractère d'intérêt. Un caractère X est une application :

Ω -->χ

ω --> X

χ est l'ensemble où X prend ses valeurs

Example :

1. On peut s'intéresser à : l'âge, poids, taille, sexe, série du bac, mention du bac des étudiants.

2. On peut s'intéresser à : la couleur, l'année de production, le kilométrage des véhicules

Selon les exemples précédents, on peut remarquer que χ peut être de nature différente (mesurable, non mesurable, dénombrable ou continu), donc on peut distinguer deux types de caractères.

Si χ et ses valeurs ne sont pas mesurables, le caractère est dit qualitatif.

Example :

1. Mention du bac d'un étudiant, série du bac ;

2. Couleur du véhicule, groupe sanguin d'un malade .

Si χ et ses valeurs sont mesurables, le caractère est dit quantitatif.

Un caractère quantitatif est dit discret si ses valeurs sont dénombrables :

Example :

1. Le nombre d'enfants par ménage ;

2. La note d'un examen.

Un caractère quantitatif est dit continu s'il peut prendre n'importe quelle valeur dans un intervalle donné de R

Example :

1. Moyenne du bac ;

2. Taille d'un étudiant.

2.2/ Modalités

Les modalités sont les différentes valeurs que peut prendre le caractère X.

Si le caractère est quantitatif discret, on note X = {x1; x2; ...; xk}, où les xi désignent les différentes valeurs possibles que peut prendre X, i.e les différentes modalités. Les xi sont supposés ordonnés x1 < x2<.... < xk.

Example :

1. Les modalités du caractère "nombre d'enfants par ménage" sont : 0, 1, 2, 3,. . .

2. Les modalités du caractère "note d'un examen" sont : 5, 8.75, 10, 11.25, 11.5,. . .
