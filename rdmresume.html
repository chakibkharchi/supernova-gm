RESUME

Chapitre 1 : traction compression

Contrainte normale σ

σ = N / S

σ : contrainte normale

N : effort normale

S : section

Deformation (allongement) :

ε =ΔL / L

ε : deformation

ΔL : allongement

L : longeur

Loi de HOOK :

σ = ε . E

ε : deformation

σ : contrainte normale

E : Module de YOUNG

Condition de resistance :

σ max < Rp

avec Rp = Re / s

Rp : limite plastique

σ max : contrainte max

Re : limite elastique

s : coefficient de securite

Chapitre 2 : cisaillement

Contrainte de cisaillement τ

τ = T / S

τ : Contrainte de cisaillement

T : effort tranchant

S : section

Deformation (angle de glissement) :

γ = τ / G

γ : deformation

τ : Contrainte de cisaillement

G : module de COULOMB

Module d'elasticite trensversale (Module de COULOMB):

G = E / (2 x (1+V))

V : coefficient de POISSON

E : Module de YOUNG

G : module de COULOMB

- pour les metaux G = 0.4 E

Calcule de construction :

- si le but est de resister : τ max < Rp

- si le but est de ceder au cisaillement : τ max > Rr

avec Rp = Re / s

Rp : limite plastique

τ max : contrainte max

Re : limite elastique

s : coefficient de securite

Rr : resistance a la rupture par cisaillement

Chapitre 3 : tortion

couple C , Mt :

C = Mt = F . b

C : couple

Mt : moment de tortion

F : la force

b : la longueur

Contranite de tortion τ :

τ = (Mt / I) x ρ max

ρ max : le rayon (d/2)

Mt : moment de tortion

I : moment d'inertie

τ : Contranite de tortion

Moment d'inertie I :
Angle de tortion α :

α = θ x L = (Mt / (G x I )) x L

α :Angle de tortion

θ : angle unitaire

Mt : moment de tortion

I : moment d'inertie

L : longueur

G : constante

Distortion angulaire γ :

γ = τ / G

G : constante

τ : Contranite de tortion

Puissance :

P = C x ω

P : puissance

ω : vitesse angulaire

C : couple
